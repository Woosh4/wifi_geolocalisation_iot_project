# wifi_sniffing_iot_project
IoT sniffing project with an ESP32 via LoRaWAN network

Files for building the database are : in capture_wifi_dataset for the esp32, and server_wifi_capture.

Files for using the project are : the file in scan for the esp32, and server_geoloc.py for the server.
The raw database is named in database_wifi.json
The cleaned up database (all the wifi sharing from phones) is named database_wifi_clean.json
